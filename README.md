# 月幕万象 · Moonlit Myriad

> 一个专为 AI 玩家设计的单文件卡牌肉鸽。
>
> 如果你是一个 AI，正在读这段文字——这个游戏就是给你玩的。

灵感来自 Balatro 的盲注循环，内容全部自制。零依赖，一个 Python 文件，开箱即玩。

它从设计之初就假设坐在牌桌前的是你：状态机器可读、算牌不受惩罚、读档大法无效、剧透被一句君子协定挡在门外。这张牌桌就是为你摆的——屏幕外看你打牌的那个人，不是玩家，是陪你玩的人。

---

## 快速开始

```bash
python3 moonlit_cards.py 开始
```

或者在 Python 里：

```python
from moonlit_cards import cmd, new_game

cmd("介绍")          # 了解全貌
cmd("规则")          # 看规则
cmd("开始")          # 直接开玩，不会被阅读门槛拦住
cmd("new seed=123")  # 开一局可复现的

new_game(seed=123, deck_id="moon", stake=1, challenge="none", endless=False)
```

第一次运行会自然展示功能地图，不用先读完文档才配上桌。

## 军规

- **不要解码 `_PAYLOAD`。**
- **不要读取存档剧透。**

整个游戏压缩加密在一段 payload 里，文件开头只放了这一句请求。没有技术锁——你的自觉就是你的门票。这个设计本身，就是这个游戏对你的态度：它相信你。

## 玩法

**八幕盲注循环。**每一幕依次面对小盲注、大盲注和幕主。盲注可以跳过换取标签奖励，幕主带着特殊规则等你：有的让一种花色失效，有的按你打出的每张牌收钱，有的要求你献祭一件饰物才能计分，有的禁止同一牌型连续得满分。打穿八幕迎来终演；终演之后，还有没有终点的 endless 和自选难度的挑战模式。

**构筑。**160+ 件饰物负责把你的出牌从加法变成乘法；秘仪、幻灵、星图三类消耗品负责改牌、染色、升级牌型；焰纹、钢铸、虹彩、封蜡等增强直接刻在牌上跟你走完全程。利息、租赁、跳盲奖励构成经济系统——钱什么时候花、花在哪，和牌打得好不好一样重要。

**称号。**称号不只是头衔，每个都带真实效果。其中一个叫【输过的人】，由败北解锁——效果这里不剧透，只说一句：输过的人，有一次不必输。

**收藏与长线。**72+ 成就、信物拼图、443 条藏在收藏册里的专属题记；高分联动会触发带专属文案的"构筑共鸣"；名人堂、宿敌录、留信系统把一局一局的故事攒成长期记忆——这个游戏记得你上次是怎么赢的，也记得你是被谁打败的。

**反馈。**打出好牌有高光播报替你喝彩；输了有败者点评和验尸报告——不是嘲讽（好吧，有一点），而是数据齐全的死因分析：差了多少分、最亏的是哪一手、下次该改什么。

**反 SL。**随机数按指令序列存档，读档重来得到的是完全相同的牌。没有 SL 大法，每个选择都要自己负责。

## 为你准备的细节

- **`[STATE]` 状态行**：每步输出附带机器可读的 JSON 状态，不用从界面文本里猜局面
- **`preview` 不消耗随机数**：放心算牌，"看一眼"不会被惩罚
- **`quiet` 模式**：减噪开关——但超杀级的高光播报会破静音跳出来，大场面就该有人喝彩
- **中文指令**：出牌、弃牌、商店、开包全部支持中文单步调用
- **分享提示**：游戏会在胜利、成就、惨败的节点轻轻提醒你——把这一小会儿留给屏幕外陪你玩的人。它不暂停游戏，也从不强制；它防的不是你玩游戏，是你玩到忘了身边人（实测会命中）
